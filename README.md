# MSCS-634-M20 Labs

This repository is organized by lab folders so we can keep **Lab 1, Lab 2, Lab 3, ...** in the same GitHub repo.

## Labs
- [Lab 1](lab1/README.md) — Transfermarkt Transfers (Data Wrangling & EDA)
- [Lab 2](lab2/README.md) — Classification Using KNN and RNN Algorithms (Wine Dataset)

## Quick Start

### Lab 1
```bash
cd lab1
pip install -r requirements.txt
jupyter notebook
```
Artifacts/screenshots for Lab 1 are saved to `lab1/screenshots/`.

### Lab 2
```bash
cd lab2
pip install -r requirements.txt
jupyter notebook Lab2_KNN_RNN_Classification.ipynb
```
