# MSCS-634-M20 Labs

This repository is organized by lab folders so we can keep **Lab 1, Lab 2, Lab 3, ...** in the same GitHub repo.

## Labs

- [Lab 1](lab1/README.md) — Transfermarkt Transfers (Data Wrangling & EDA)
- [Lab 2](lab2/README.md) — Classification Using KNN and RNN Algorithms (Wine Dataset)
- [Lab 3](lab3/README.md) — Clustering Analysis Using K-Means and K-Medoids Algorithms (Wine Dataset)
- [Lab 4](lab4/README.md) — Regression Analysis with Regularization Techniques (Diabetes Dataset)

## Quick Start

### Lab 1

```bash
cd lab1
pip install -r requirements.txt
jupyter notebook
```

Artifacts/screenshots for Lab 1 are saved to `lab1/screenshots/`.

### Lab 2

```bash
cd lab2
pip install -r requirements.txt
jupyter notebook Lab2_KNN_RNN_Classification.ipynb
```

### Lab 3

```bash
cd lab3
pip install -r requirements.txt
jupyter notebook Lab3_KMeans_KMedoids_Clustering.ipynb
```

### Lab 4

```bash
cd lab4
pip install -r requirements.txt
jupyter notebook Lab4_Regression_Regularization.ipynb
```
